# 플로로탄닌 파트너스 (Phlorotannin Partners)

해양 폴리페놀 플로로탄닌 전문 건강 정보 커뮤니티

---

## ⚠️ AI 개발자 필독 (작업 시작 전 반드시 읽을 것)

이 프로젝트는 **고정된 작업 규칙**을 따른다. 신규 AI 개발자는 작업 전 아래 3개 문서를 반드시 먼저 읽고, 사용자에게 작업 계획부터 보고한다.

1. [`AI_WORK_RULES.md`](./AI_WORK_RULES.md) — 작업 헌법 (작업 전 판단, 보고 형식, 검증 방식)
2. [`PROJECT_MAP.md`](./PROJECT_MAP.md) — 프로젝트 구조 지도 (어디를 봐야 하는지)
3. [`DO_NOT_TOUCH.md`](./DO_NOT_TOUCH.md) — 절대 건드리지 말 것 목록

**사용자가 승인하기 전에 코드나 DB를 절대 수정하지 않는다.**

## 📋 주요 기능
- 1,200개 건강 Q&A (12개 카테고리)
- 플로로탄닌 전문 소개 페이지 (6가지 작용 기전)
- 파트너 연결 프로그램
- 1:1 건강 상담

## 🛠 기술 스택
- React + Vite
- Tailwind CSS
- Vercel 배포
